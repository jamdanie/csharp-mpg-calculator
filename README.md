C# MPG Calculator

A simple C# console application that calculates Miles Per Gallon (MPG) based on user input for miles driven and gallons of fuel used.

This project was created as part of coursework to practice basic input/output, arithmetic operations, and program structure in C#.

Features

Prompts the user to enter:

Miles driven

Gallons of fuel used

Calculates Miles Per Gallon (MPG)

Displays the calculated fuel efficiency

Demonstrates basic console interaction

Formula Used

The MPG calculation uses the formula:

MPG = Miles Driven / Gallons Used

Example:

Miles Driven: 300
Gallons Used: 10

MPG = 300 / 10
MPG = 30 miles per gallon
